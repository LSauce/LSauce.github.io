安装node.js

安装包：官网
可使用cnpm淘宝镜像

安装Hexo

  • 进入git终端,全局安装hexo
    cnpm install -g hexo-cli

    -g代表全局安装，hexo后的 -cli 代表命令行
    也就是说再安装hexo的同时也会安装hexo的命令行

  • 初始化hexo模板
    hexo init

  • 启动hexo静态博客

    hexo server 可用 hexo s 代替

  • 访问http://127.0.0.1:4000可进入页面

将博客托管到Github page

  • 在GitHub新建库 命名为 用户名.github.io

  • 安装hexo-deployer-git插件
    cnpm install hexo-deployer-git --save

  • 配置hexo博客根目录中的_config.yml 在最后添加

  • 本地生成静态文件
    hexo g

  • 将静态文件推送至Github

    hexo d

  • 访问博客
    http://用户名.github.io
    可能需等几分钟才能访问
    如果想国内访问更流畅可使用vercel托管Github page
